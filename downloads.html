How to use a downloaded file. Save it anywhere, then double-click to open it in your web browser. The bench calculators work with no connection at all. The reference formatter and QueryBee need the internet only to look up reference details. To share a tool, just send the file; to host your own copy, put the files on any static web host. See the about page for steps.
